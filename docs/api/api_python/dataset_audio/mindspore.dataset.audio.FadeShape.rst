mindspore.dataset.audio.FadeShape
=================================

.. py:class:: mindspore.dataset.audio.FadeShape

    淡入淡出形状。

    可能的枚举值包括： ``FadeShape.QUARTER_SINE`` 、 ``FadeShape.HALF_SINE`` 、 ``FadeShape.LINEAR`` 、 ``FadeShape.LOGARITHMIC`` 、 ``FadeShape.EXPONENTIAL``。

    - **FadeShape.QUARTER_SINE**：表示四分之一正弦模式。
    - **FadeShape.HALF_SINE**：表示半正弦模式。
    - **FadeShape.LINEAR**：表示线性模式。
    - **FadeShape.LOGARITHMIC**：表示对数模式。
    - **FadeShape.EXPONENTIAL**：表示指数模式。
